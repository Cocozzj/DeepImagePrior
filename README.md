
# Install

Here is the list of libraries you need to install to execute the code:
- python = 3.6
- [pytorch](http://pytorch.org/) = 0.4
- numpy
- scipy
- matplotlib
- scikit-image
- jupyter


or create an conda env with all dependencies via environment file

```
conda env create -f environment.yml
```
